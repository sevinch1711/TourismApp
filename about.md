# Buxoro Digital Tourism (Smart Tourism Platform)

Ushbu loyiha Buxoro shahrining qadimiy tarixi va boy madaniyatini zamonaviy raqamli texnologiyalar yordamida targ'ib qilish uchun yaratilgan "Aqlli Turizm" (Smart Tourism) platformasidir. Loyiha sayyohlar uchun virtual va immersiv tajriba taqdim etishga qaratilgan bo'lib, quyidagi yutuqlarni o'z ichiga oladi:

## Resume/CV uchun matn (DigiLab formatida)

*   **Engineered an interactive Smart Tourism platform** featuring 20+ historical monuments of Bukhara, utilizing React, TypeScript, and modern WebGL technologies for 360° virtual tours.
*   **Integrated an AI-powered Trip Planner and Interactive Map (Leaflet)** with dynamic Haversine distance calculations to provide personalized itineraries and nearest-site diagnostics based on predefined user preferences (Sufism, History, Photography).
*   **Link:** https://tourismapp-blond.vercel.app/

---

## Qo'shimcha ma'lumot (Loyiha haqida kengroq)

**Asosiy Funksionalliklar:**
1.  **360° Virtual Sayohat (VR):** Foydalanuvchilar qadimiy obidalarni (masalan, Yetti Pir maqbaralari) kompyuter yoki telefon orqali butunlay 3D muhitda aylanishi mumkin.
2.  **Multilingual (Ko'p tilli) tizim:** Sayt O'zbek, Rus va Ingliz tillarida to'liq ishlaydi.
3.  **Kengaytirilgan Xarita (Interactive Map):** Leaflet asosida qurilgan maxsus xarita. Unda foydalanuvchining joylashuvi paneli joylashgan bo'lib, sayyohning joriy nuqtasidan eng yaqin obidalargacha bo'lgan masofani aniq hisoblab beradi.
4.  **AI Virtual Gid / Trip Planner:** Sayohatchining qiziqishi, byudjeti va vaqtini inobatga olgan holda maxsus rejalar tuzib beruvchi aql markazi.

**Texnologik Stack:**
*   **Frontend:** React (v19), TypeScript, Tailwind CSS
*   **Xarita va Interfaol qismlar:** React-Leaflet, Lucide-React
*   **Build/Deploy:** Vite, Vercel
